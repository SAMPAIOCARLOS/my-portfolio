HABILIDADES

HTML5

O HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem fundamental da web. Foi criado por Tim Berners-Lee no início da década de 1990 como parte do projeto World Wide Web (WWW) no CERN, na Suíça. Desde então, o HTML evoluiu e se tornou a base para a construção de páginas da web e aplicativos da web.

CSS3

O Cascading Style Sheets, mais conhecido como CSS, é uma linguagem de marcação amplamente utilizada na criação e formatação de páginas da web. É uma parte fundamental das tecnologias da web.

JAVASCRIP

JavaScript é uma linguagem de programação de alto nível, dinâmica e amplamente utilizada para desenvolvimento web e além. Ela desempenha um papel fundamental na criação de páginas web interativas e dinâmicas, adicionando funcionalidade e comportamento aos sites.

GIT-HUB

O GitHub é uma plataforma online amplamente utilizada para hospedar, compartilhar e colaborar em projetos de desenvolvimento de software. Fundado em 2008, o GitHub se tornou uma ferramenta fundamental para desenvolvedores, equipes de desenvolvimento, empresas e comunidades de código aberto.
